Cargar Archivo Excel

Guía para preparar el archivo

Requisitos del archivo:

  • Formato Excel XLSX o CSV
  • Tamaño máximo: 5MB
  • Hasta 200,000 filas
  • Primera fila debe contener los nombres de las columnas
  • Fecha en formato dd-MM-yyyy o dd/MM/yyyy
  • Valores numéricos usando punto como separador decimal
Nota: Una vez cargado el archivo, podrá seleccionar las columnas específicas requeridas para el tipo de servicio elegido.
