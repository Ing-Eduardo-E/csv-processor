Procesador de Archivos CSV

Cargar Archivo CSV

Guía para preparar el archivo

Requisitos del archivo:

  • Formato CSV (valores separados por comas)
  • Tamaño máximo: 5MB
  • Hasta 200,000 filas
  • Codificación UTF-8
  • Archivo completo del sistema de acueducto (todas las columnas)
Nota: La aplicación procesará automáticamente el archivo completo del sistema. No es necesario eliminar columnas ni modificar el formato original.

Columnas requeridas del archivo original:

Columna del Sistema Descripción
FECHA DE EXPEDICIÓN DE LA FACTURA Fecha de facturación
CÓDIGO CLASE DE USO Clasificación del usuario
ESTADO DE MEDIDOR INSTALADO, NO INSTALADO, etc.
CONSUMO DEL PERÍODO EN METROS CÚBICOS Consumo medido
VALOR TOTAL FACTURADO Total facturado al usuario
PAGOS DEL USUARIO RECIBIDOS DURANTE EL MES DE REPOPRTE Recaudos del período

💡 Tip: Simplemente exporte el archivo CSV completo desde su sistema de facturación. La aplicación se encargará de procesar y resumir la información automáticamente.
